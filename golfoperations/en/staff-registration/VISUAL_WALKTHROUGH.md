## 🔴 Common Error Messages

### Error 1: Invalid Code

```
╔════════════════════════════════╗
║  ❌ Verification Failed        ║
╠════════════════════════════════╣
║                                ║
║  Invalid golf course           ║
║  registration code             ║
║                                ║
║  Please check with your        ║
║  manager and try again.        ║
║                                ║
║  ┌───────────────────────────┐ ║
║  │          OK               │ ║
║  └───────────────────────────┘ ║
╚════════════════════════════════╝
```

**Solution**: Double-check 4-digit code with manager

---

### Error 2: Invalid Employee ID Format

```
╔════════════════════════════════╗
║  ❌ Verification Failed        ║
╠════════════════════════════════╣
║                                ║
║  Invalid Employee ID format    ║
║  Expected: PAT-001             ║
║                                ║
║  Your format: pat001           ║
║                                ║
║  Use: [PREFIX]-[NUMBERS]       ║
║  Example: PAT-023              ║
║                                ║
║  ┌───────────────────────────┐ ║
║  │          OK               │ ║
║  └───────────────────────────┘ ║
╚════════════════════════════════╝
```

**Solution**: Use correct format with dash and uppercase

---
