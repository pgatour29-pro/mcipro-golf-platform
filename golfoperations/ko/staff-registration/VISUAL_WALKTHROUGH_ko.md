# 시각적 안내 - 직원 등록 프로세스

## 스크린샷과 함께하는 완전한 등록 과정

---

## 🎯 개요

**소요 시간**: 5-10분
**난이도**: 쉬움
**요구사항**: LINE 앱이 설치된 스마트폰

---

## 📱 단계별 시각적 가이드

### 1단계: 자격 증명 받기

**관리자로부터 받을 정보:**

```
┌─────────────────────────────────────┐
│     직원 등록 정보                  │
├─────────────────────────────────────┤
│                                     │
│  골프장 코드:  1234                 │
│  직원 ID:      PAT-023              │
│  부서:         캐디                 │
│                                     │
│  이 정보를 안전하게 보관하십시오!   │
└─────────────────────────────────────┘
```

---

### 2단계: 인증 페이지 열기

**방법 A: QR 코드** (가장 쉬움)
```
     ┌────────────────┐
     │  [QR CODE]     │
     │  ████████████  │
     │  ██  ████  ██  │
     │  ████████████  │
     │                │
     │ 카메라 앱으로  │
     │ 스캔하세요     │
     └────────────────┘
```

**방법 B: URL 입력**
```
🌐 https://mcipro-golf-platform.netlify.app/staff-verification.html
```

## 🔴 일반적인 오류 메시지

### 오류 1: 유효하지 않은 코드

```
╔════════════════════════════════╗
║  ❌ 인증 실패                  ║
╠════════════════════════════════╣
║                                ║
║  유효하지 않은 골프장          ║
║  등록 코드                     ║
║                                ║
║  관리자에게 확인하고           ║
║  다시 시도하십시오.            ║
║                                ║
║  ┌───────────────────────────┐ ║
║  │          확인             │ ║
║  └───────────────────────────┘ ║
╚════════════════════════════════╝
```

**해결책**: 관리자에게 4자리 코드를 다시 확인

---

╠════════════════════════════════╣

║  예시: PAT-023                 ║

## 🎯 성공 체크리스트

각 단계를 확인하는 데 사용:

```
등록 진행 상황 추적기

□ 1단계: ✓ 관리자로부터 골프장 코드 받음
□ 2단계: ✓ 직원 ID 할당 받음
□ 3단계: ✓ 부서 알고 있음
□ 4단계: ✓ LINE 앱 설치됨
□ 5단계: ✓ 인증 페이지 열림
□ 6단계: ✓ 골프장 코드 올바르게 입력됨
□ 7단계: ✓ 부서 선택됨
□ 8단계: ✓ 직원 ID 입력됨 (형식 정확함)
□ 9단계: ✓ 인증 통과됨
□ 10단계: ✓ LINE 로그인 완료됨
□ 11단계: ✓ 프로필 정보 입력됨
□ 12단계: ✓ 프로필 저장됨
□ 13단계: ✓ 확인 받음
□ 14단계: ✓ 승인 받음 (필요한 경우)
□ 15단계: ✓ 대시보드 액세스됨
□ 16단계: ✓ 대시보드 기능 탐색됨

✅ 등록 완료!
```
